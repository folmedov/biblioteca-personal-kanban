# Sistema de Biblioteca Personal

**Contexto**

Un **dueño de libros** quiere prestar un ejemplar de su colección a un **amigo**. Primero, busca el **libro** en su estantería para verificar que esté disponible. Una vez que lo tiene, registra la entrega indicando quién es el **prestatario** y en qué **fecha** se realiza el préstamo. El sistema confirma que el libro ahora está 'prestado'. Semanas después, cuando el amigo devuelve el libro, el dueño registra la **devolución**, el libro vuelve a estar 'disponible' y queda un **historial** de quién lo tuvo.

## Objetivos

### Objetivo general:

- Crear un sistema de gestión de prestamos de libros

### Objetivos específicos:

- Llevar un registro de todos los libros en la biblioteca
- Llevar un registro de cada préstamo realizado por un socio
